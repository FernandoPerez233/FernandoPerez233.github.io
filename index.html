Esta página ha sido creada como parte de un proyecto escolar. Su objetivo principal es servir como espacio de práctica para aplicar los conocimientos que estoy adquiriendo en el desarrollo web. A lo largo del sitio, iré desarrollando diferentes secciones que mostrarán tanto mi progreso como las habilidades que voy mejorando con el tiempo. Entre las secciones que incluiré están: una presentación personal, tablas informativas (como un horario y calificaciones), enlaces a redes sociales, y posiblemente un portafolio con proyectos sencillos. Todo esto me ayudará a entender mejor el uso de etiquetas HTML, estilos con CSS y la estructura general de una página web. Este proyecto no solo me permite cumplir con una actividad escolar, sino que también me motiva a seguir aprendiendo, explorando nuevas ideas, y creando contenido de forma organizada y visualmente atractiva.
