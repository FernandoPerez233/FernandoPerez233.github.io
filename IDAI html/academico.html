Para hacer que un archivo sea descargable en una página HTML, se utiliza la etiqueta < a >
junto con el atributo href , que sirve para indicar la ubicación del archivo dentro del sitio.
Por ejemplo, se puede escribir algo como < a href="ruta-del-archivo" download >.
El atributo download es el que le dice al navegador que, en lugar de abrir el archivo,
debe descargarlo cuando el usuario haga clic en el enlace. De esta forma, se puede facilitar al visitante la descarga directa de documentos,
imágenes u otros tipos de archivos desde la página web.
